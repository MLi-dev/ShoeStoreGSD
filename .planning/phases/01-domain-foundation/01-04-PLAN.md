  <how-to-verify>
1. Open a terminal at the project root:
   ```
   cd /Users/matthew/Dev/ShoeStoreGSD
   ```

2. Start uvicorn in the foreground:
   ```
   uv run uvicorn main:app --port 8001
   ```

3. Observe the startup log. Expected output (lines in any order, timestamps will vary):
   ```
   INFO:     Started server process [NNNN]
   INFO:     Waiting for application startup.
   INFO:     Application startup complete.
   INFO:     Uvicorn running on http://127.0.0.1:8001 (Press CTRL+C to quit)
   ```

4. Confirm that NO Python traceback appears (no lines starting with `Traceback`, no `Error:` lines, no `ImportError`, no `AttributeError`).

5. (Optional) In a second terminal, hit the root:
   ```
   curl -sS http://127.0.0.1:8001/openapi.json | head -c 200
   ```
   Expected: a short JSON prefix beginning with `{"openapi":"3.` — confirms FastAPI is serving. A 404 on `/` is expected because no routes are registered in Phase 1.

6. Stop uvicorn with CTRL+C. Expected shutdown lines:
   ```
   INFO:     Shutting down
   INFO:     Waiting for application shutdown.
   INFO:     Application shutdown complete.
   ```

7. Record the result in 01-04-SUMMARY.md:
   - PASS: "uvicorn startup complete, no traceback, shutdown clean"
   - FAIL: paste the traceback and mark the plan as blocked pending fix
  </how-to-verify>
