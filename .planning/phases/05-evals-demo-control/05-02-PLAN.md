| Dataset files → test runner | Python module import; no external I/O, no user input |
| Smoke runner → Anthropic API | Real API calls with ANTHROPIC_API_KEY credential (Plan 03 scope) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-06 | Information Disclosure | Adversarial case input strings | accept | Dataset files contain intentionally adversarial strings (injection attempts, off-topic requests) as test fixtures — they are not executable and not user-controlled. No PII, no secrets. |
| T-05-07 | Tampering | CASES lists mutated at import time | accept | Lists are module-level constants; test runner imports and reads only. No mutation path in dataset files themselves. |
| T-05-08 | Denial of Service | Very large expected_trajectory lists | accept | Lists are static, bounded, defined by developers — no runtime growth possible. |
